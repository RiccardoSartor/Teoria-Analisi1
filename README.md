# 📘 Raccolta di Teoria di Analisi 1 in LaTeX

Questo repository contiene una raccolta completa di domande degli anni precedenti riguardo esami **Analisi 1** numerate in base alla loro frequenza di apparizione in questi.
Redatto in **LyX**.  
L'obiettivo è fornire un riferimento ben organizzato per **Ingegneria Informatica** a Padova.

## 📖 Contenuti
- ✅ Domande e risposte degli esami passati

## 🛠 Come Usarlo
   **Scarica** direttamente il file .PDF
   OPPURE
1. **Clona** la repository
2. **Compila** il file principale con LyX editor
